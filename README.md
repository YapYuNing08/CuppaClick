## 📌 Project Overview

**CuppaClick** is a web-based food ordering system that allows users to browse menus, place orders, and manage food purchases online.  
The platform also includes an **admin interface** to manage menu items and customer orders.

This project demonstrates **full-stack web development using Django**, dynamic templates, and database integration.

---

## 🚀 Features

### 👤 User Features
- Browse available food and beverage menu
- View item descriptions and prices
- Add food items to order
- Place orders through the platform
- View order details

### 🔧 Admin Features
- Add, update, and delete menu items
- Manage food and beverage listings
- View and manage customer orders
- Update order status
